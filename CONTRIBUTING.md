# Contributing to AlphaCPSTest

Thank you for helping improve **AlphaCPSTest.com**! 🎉

This repository is **documentation-only** — it does not contain the website's source code. Because of that, contributions here are not code pull requests. Instead, this is the place to **report bugs, request features, and share feedback** that help shape the product.

---

## 🐞 Reporting a Bug

Found something broken on the website — a tool that won't load, an inaccurate result, a layout issue, or a broken link?

1. Check the [existing issues](../../issues) to see if it's already reported.
2. If not, [open a new bug report](../../issues/new/choose) using the **Bug Report** template.
3. Include as much detail as you can: the tool/page, your browser and device, and what you expected vs. what happened.

The more specific you are, the faster it can be fixed.

---

## 💡 Requesting a Feature

Have an idea for a new tool, a guide topic, or an improvement to an existing tool?

1. Look at the [ROADMAP.md](ROADMAP.md) — it may already be planned.
2. Check [existing issues](../../issues) for duplicates.
3. [Open a feature request](../../issues/new/choose) using the **Feature Request** template.

Explain what you'd like and why it would be useful — real use cases help prioritize.

---

## 🗣️ General Feedback

Not a bug or a specific feature? General thoughts, usability feedback, or questions are also welcome:

- [Open an issue](../../issues/new) and describe your feedback, or
- Use the contact options on the [website](https://alphacpstest.com/contact).

---

## ✅ Good Reports Include

- **What** happened (or what you'd like)
- **Where** — which tool or page (include the URL when possible)
- **How** to reproduce, for bugs (step by step)
- **Environment** — browser, operating system, device type
- **Screenshots** — if they help illustrate the issue

---

## 🚫 What This Repo Is *Not* For

- This is **not** an open-source codebase — there is no source code to modify here.
- Please do **not** submit code pull requests; they cannot be merged into the live site from here.
- For security concerns, please use the private contact options on the website rather than a public issue.

---

## 🤝 Code of Conduct

Please be respectful and constructive. Reports and requests of all kinds are appreciated, and a friendly tone keeps the community welcoming for everyone.

---

Thanks again for contributing to making AlphaCPSTest better! ⚡
