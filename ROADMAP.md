# Roadmap

This document outlines the current state of **AlphaCPSTest.com** and the planned direction of the project. Priorities may shift based on user feedback — [feature requests](../../issues/new/choose) are welcome.

---

## ✅ Current Features

### CPS / Clicking Tools
- Spacebar Counter
- Kohi Click Test
- Jitter Click Test
- Drag Click Test
- Double Click Test
- Spacebar Hold Test

### Gaming Tools
- Aim Trainer
- Reaction Time Test
- Scroll Speed Test
- Mouse Accuracy Test

### Device Testing Tools
- Camera Test
- Microphone Test
- Keyboard Test
- Typing Speed Test

### Alpha Tools
- Alpha Chess (in-browser AI opponent)

### Content
- 16 in-depth guides across 4 topic clusters
- Dedicated Articles Hub
- Full FAQ coverage

### Platform
- Clean URLs and complete sitemap
- Mobile-responsive across all tools
- Privacy-first device tests (no data upload)

---

## 🚧 Planned Features

These are under consideration or actively planned. Order does not imply timeline.

### New Tools
- [ ] **Keyboard Ghosting Test** — dedicated key-rollover and anti-ghosting checker (press multiple keys to see which register simultaneously)
- [ ] **Mouse Polling Rate Test** — measure your mouse's report rate in Hz (125 / 500 / 1000+)
- [ ] **Mouse Tester** — all-in-one mouse diagnostic: buttons, scroll, double-click detection, and movement tracking
- [ ] **Click Accuracy / Target Tracker** — extended precision modes
- [ ] **Color / Display Test** — screen dead-pixel and color checks
- [ ] **Audio Output Test** — speaker / headphone channel verification

### Tool Enhancements
- [ ] **Leaderboards** — global rankings for CPS and gaming tools
- [ ] **Score history** — track personal progress over time
- [ ] **Additional test durations** — more timed modes (1s / 5s / 10s / 30s / 60s / 100s) across CPS tools
- [ ] **Real-time performance graphs** during tests
- [ ] **Performance rank labels** (e.g. Beginner → Legend)

### Content & Guides
- [ ] **Additional guides** — deepen every existing topic cluster
- [ ] Benchmark reference pages (e.g. average CPS by skill level)
- [ ] Comparison and "how-to" articles for each new tool
- [ ] Expanded device-troubleshooting library

### Platform & Experience
- [ ] Multi-language support
- [ ] Dark/light theme options
- [ ] Accessibility improvements (keyboard navigation, screen-reader labels)
- [ ] Continued Core Web Vitals / performance optimization
